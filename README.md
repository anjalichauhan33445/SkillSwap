# SkillSwap - Peer-to-Peer Skill Exchange Platform

A full-stack web application that allows users to exchange skills and connect with others.

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoBB

## Features
- User authentication (login/register)
- Secure password handling using bcrypt
- Session-based authentication
- Dynamic dashboard using EJS

## How to Run
1. Clone the repository
2. Run npm install
3. Start server using node app.js
